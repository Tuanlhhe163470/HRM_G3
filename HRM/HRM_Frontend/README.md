This is HRM System. This project used for SWP391 course in FPT UNIVERSITY. Developed by Group3

## Instruction run project
First, clone project. After that run ``` npm i```` to install library:
Second run
```bash
npm run dev
```
Wait and click on link [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Project structure

Project structure include:
-auth
-hrm
-public
-components
-constants
-contexts
-hooks
-redux
-services
-theme
